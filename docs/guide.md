# UsesKit <!-- {docsify-ignore-all} -->
> 集成了JS组件，来提高你的开发速度

### JavaScript 组件
* 简单易用并且体积轻量
* 无需构建，本地引用或CDN引用
* 低耦合，最大化保持组件之间独立运行
* 所有组件都是基于原生`javascript`开发
#### 组件列表
   * KitJS
   * KitRouter

### NodeJS 组件
* 快速构后端API
* 自动生成路由
* 自动生成`create`, `read`, `update`, `delete`, `info`, `list`, `list-page`常用操作
#### 组件列表
   * eslint-config
   * Midway-crud
   * midway-qiniu
   * midway-casbin
   * midway-log2db
